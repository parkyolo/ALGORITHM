## 멀리 뛰기
### 문제
https://programmers.co.kr/learn/courses/30/lessons/12914
### 풀이
- 1칸일 때 1                                                                  -> 1가지
- 2칸일 때 1 1, 2                                                             -> 2가지
- 3칸일 때 1 1 1, 1 2, 2 1                                                    -> 3가지
- 4칸일 때 1 1 1 1, 1 1 2, 1 2 1, 2 1 1, 2 2                                  -> 5가지
- 5칸일 때 1 1 1 1 1, 1 1 1 2, 1 1 2 1, 1 2 1 1, 2 1 1 1, 1 2 2, 2 1 2, 2 2 1 -> 8가지  
  
가짓수가 피보나치 수열로 증가한다.
