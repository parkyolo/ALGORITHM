## 별
### 문제
https://www.acmicpc.net/problem/16505  
출력 예제를 보고 별 찍는 규칙을 유추하여 별을 찍어 보자.

### 풀이
- 출력의 가로, 세로 길이는 2의 N승이다.
- (0, 0)의 위치에는 무조건 별이 찍히기 때문에 (0, 0)에서 시작한다.
- n = 2**N일 때 별이 찍히는 위치를 찾으면
    - nn = n/2
    - (x, y)
    - (x + nn, y)
    - (x, y + nn)
    - 이렇게 3개의 위치에 별이 찍힌다.
- 재귀적으로 별을 찍어준다.
